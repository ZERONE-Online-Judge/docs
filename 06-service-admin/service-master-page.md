# 서비스 마스터 페이지

## 역할

서비스 마스터는 시스템 전체 운영을 총괄하는 최상위 권한 사용자다.

## 주요 메뉴

- 대회 생성 / 오픈 / 종료 / 삭제
- 서비스 공지 게시판 관리
- 서비스 매니저 관리
- 대회 운영자 관리
- 전체 참가 팀 관리
- 전체 문제/제출/감사 로그 조회
- 채점 서버/클러스터 설정

## 페이지에서 다뤄야 하는 것

- 전체 대회 현황
- 현재 진행 대회 상태
- 채점 인프라 상태
- 운영 인력 현황
- 서비스 공지 게시판 상태
- 전역 긴급 공지 발행 상태

## 핵심 요구사항

- 서비스 마스터는 모든 대회 생성/오픈 권한을 가진다.
- 서비스 매니저도 `contest.create`, `contest.open` 등 명시적으로 부여받은 permission 범위 안에서는 대회 생성/오픈 작업을 수행할 수 있다.
- 서비스 마스터는 모든 리소스에 접근 가능
- 민감 작업은 감사 로그로 남아야 함
