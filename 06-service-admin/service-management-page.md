# 서비스 관리 페이지

## 목적

서비스 단위 설정과 운영 리소스를 관리하는 페이지다.
서비스 마스터가 주 사용자이며, 일부 기능은 권한 있는 서비스 매니저가 접근할 수 있다.

## 포함 기능

- 서비스 매니저 목록/추가/수정/삭제
- 서비스 매니저 권한 체크리스트
- 권한 범위 설정
- 서비스 공지 게시판 관리
- 메일 템플릿 조회/수정
- 채점 서버 상태 조회
- 채점 서버 설정 변경
- 채점 클러스터 정책 관리
- 감사 로그 조회

## 서비스 매니저 관리 요구사항

- 전역 권한 또는 특정 대회 범위 권한 부여 가능
- 권한 변경 이력 관리 필요
- 초대 메일 및 비밀번호 설정 메일 발송 가능

## 인프라 관리 요구사항

- judge queue 상태 조회
- judge server 상태 조회
- slot/scale 정책 조정
- 장애 node 비활성화 처리

## 운영 원칙

- 서비스 레벨 설정 변경은 모두 감사 로그 대상이다.
- 권한 없는 사용자는 서비스 관리 페이지에 접근할 수 없어야 한다.
