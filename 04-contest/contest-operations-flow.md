# 대회 운영 플로우

## 시작 전

1. 대회 생성
2. 운영자/운영매니저 배정
3. 대회 정보 입력
4. 규정집 작성
5. 참가 팀 등록
6. 문제/리소스/테스트케이스 등록
7. 공개 시각과 시작/종료 시각 설정
8. 최종 점검 후 오픈

## 진행 중

1. 공지와 긴급 공지 운영
2. 질문 게시판 답변
3. 제출/채점 큐/스코어보드 모니터링
4. 참가 팀 상태 관리
5. 필요 시 실격 처리
6. 프리즈 시점 도달 시 공개 스코어보드 정책 전환

## 종료 후

1. 최종 결과 검토
2. 필요 시 재채점 검토
3. 최종 스코어보드 공개
4. 종료 후 문제 공개 정책 집행
5. 운영 로그와 결과 보관

## 운영 원칙

- 단계별 허용 작업이 명확해야 한다.
- 민감 작업은 감사 로그에 남아야 한다.
- 진행 중에는 대회 공정성을 깨는 수정이 불가능해야 한다.
