# 대회별 기능

## 참가자 기능

- 대회 조회
- 대회 참가 로그인
- 문제 조회
- 코드 제출
- 제출 현황 조회
- 스코어보드 조회
- 공지사항 조회
- 질문 등록 및 조회

## 운영 기능

- 대회 정보 수정
- 대회 채점 제도 설정
- 대회 점수 계산 정책 설정
- 참가 팀 관리
- 문제 관리
- 리소스/테스트케이스 관리
- 공지 작성/수정/삭제
- 질문 답변
- 스코어보드 프리즈/해제/최종 공개

## 서비스 단위 기능

- 대회 생성
- 대회 오픈/종료/삭제
- 운영자/운영매니저 배정
- 채점 서버 상태 관리
- 채점 인프라 권한 기반 재채점 요청
- 감사 로그 조회

## 대회 상태와 연결된 기능

- 시작 전에는 운영 설정과 준비 작업 중심
- 진행 중에는 참가와 모니터링 중심
- 종료 후에는 결과 검토와 공개 정책 집행 중심

## 점수제 대회 추가 기능

- 문제별 서브태스크 기준 관리
- 문제별 배점 합계 `100점` 검증
- 대회별 점수 계산 정책 설정
- 문제별 현재 획득 점수 표시
- 총점 기준 스코어보드 정렬
- 총점 달성 제출 시각(`submitted_at`) 기반 동점 처리
- 종료 후 팀/문제별 점수 개별 조정
