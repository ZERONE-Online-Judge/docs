# 대회 페이지 구성

## 대회 홈 기본 영역

- 대회 제목과 주최 정보
- 대회 상태 배지
- 시작/종료/프리즈 시각
- 대회 개요
- 참가 로그인 진입 버튼
- 공지사항 진입
- 질문 게시판 진입
- 규정집 진입

## 대회 내 주요 탭 또는 메뉴

- 홈
- 문제
- 제출 현황
- 스코어보드
- 공지사항
- 질문 게시판
- 규정집

## 상태별 페이지 차이

- 시작 전
  - 문제 비공개
  - 규정/공지 위주
- 진행 중
  - 문제/제출/스코어보드 활성
- 종료 후
  - 공개 정책에 따라 문제와 스코어보드 노출 범위 결정

## 운영자 관점 추가 요소

- 현재 참가 팀 수
- 제출량
- 채점 큐 상태
- 긴급 공지 작성 진입
- 긴급 공지 배너에서 공지사항 상세/목록으로 이동하는 링크 상태 확인
