# 대회 목록 페이지 플로우

## 목적

대회 목록 페이지는 현재 진행/예정/종료 대회를 구분해 보여주고, 각 대회 홈으로 진입시키는 페이지다.

## 목록에 포함할 정보

- 대회명
- 주최 단체
- 상태
- 공개 여부
- 시작/종료 시각
- 대회 홈 이동 링크

## 사용자 플로우

1. 사용자가 대회 목록 페이지에 진입한다.
2. 진행중, 공개예정, 종료 대회를 구분해서 본다.
3. 특정 대회를 선택한다.
4. 대회 홈에서 개요, 공지, 질문 게시판, 로그인 진입 버튼을 본다.

## 핵심 요구사항

- 현재 열려 있는 대회와 종료된 대회를 분명히 구분해야 한다.
- 공개 전 대회의 노출 정책은 대회 설정을 따라야 한다.
- 대회 선택 후 자연스럽게 대회 홈과 로그인 플로우로 이어져야 한다.
