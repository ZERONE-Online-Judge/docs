# 메인 페이지 플로우

## 목적

메인 페이지는 서비스 소개와 주요 진입점 제공이 목적이다.

## 보여줘야 하는 것

- 서비스 소개
- 대회 개최 방식 소개
- 운영 규정 안내
- 대회 생성 요청 방법
- 공개 채점 서버 현황 링크
- 현재 진행/예정 대회로 이동하는 CTA

## 사용자 플로우

1. 사용자가 메인 페이지에 진입한다.
2. 서비스 목적과 현재 운영 중인 기능을 확인한다.
3. 대회 목록 또는 특정 대회로 이동한다.
4. 필요하면 운영 규정, 대회 생성 요청 안내, 공개 인프라 현황을 확인한다.

## 핵심 요구사항

- 처음 보는 사용자도 서비스 목적을 이해할 수 있어야 한다.
- 참가자와 운영자 모두에게 유효한 공통 진입점이어야 한다.
- 메인 페이지에서 권한자 로그인과 참가 대회 진입을 혼동하지 않게 해야 한다.
