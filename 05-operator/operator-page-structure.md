# 대회 운영자 페이지 구성

## 핵심 메뉴

- 대회 개요 / 설정
- 운영 인력 관리
- 참가 팀 관리
- 문제 관리
- 리소스/테스트케이스 관리
- 제출/제출 코드/팀별 현황
- 스코어보드 관리
- 공지사항
- 질문 게시판
- 감사 로그 또는 변경 이력

## 대시보드에 보여줄 것

- 현재 대회 상태
- 참가 팀 수
- 제출량
- 채점 큐 상태
- 문제별 정답 수 / 제출 수
- 팀별 현재 순위 변화
- 긴급 공지 발행 상태

## UI 원칙

- 시작 전/진행 중/종료 후 상태가 분명히 보여야 한다.
- 금지된 작업은 버튼과 안내 문구 수준에서 명확히 차단해야 한다.
- 민감 작업은 확인 절차와 감사 로그를 남겨야 한다.
